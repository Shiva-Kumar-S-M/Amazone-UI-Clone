Amazon Clone – Frontend Project

This project is a simplified frontend recreation of the Amazon homepage, built using HTML and CSS as part of my web development learning journey.

The goal was to understand how real-world interfaces are structured, styled, and organized at a foundational level. Instead of only following tutorials, I focused on rebuilding the layout to strengthen practical understanding.

What this project demonstrates:

* Structured and semantic HTML usage
* CSS layout techniques including flexbox and grid
* Ability to replicate a real-world user interface
* Attention to spacing, alignment, and visual hierarchy

Tech stack:

HTML
CSS

Project structure:

index.html
style.css
images/

Future scope:

* Improve responsiveness across all devices
* Add interactivity using JavaScript
* Expand into a full-stack application

This project reflects my transition from learning concepts to applying them in practice.

I am continuously working on improving both code quality and design thinking.

A simple build today is a stronger foundation tomorrow.
